# Skills

Las skills viven en:

```text
.opencode/skills/<nombre>/SKILL.md
```

Cada skill debe tener frontmatter:

```yaml
---
name: nombre-skill
description: Descripción clara
compatibility: opencode
---
```

## Notas

- `frontend/SKILL.md` es la skill principal de frontend.
- La skill `frontend` cubre interfaces web, componentes UI, landing pages, widgets embebibles y plantillas de email.
- `frontend/GUIDELINES.md` actúa como referencia extensa de UI/UX, accesibilidad y performance frontend.
- `frontend/EMAIL_DESIGN.md` actúa como guía específica para newsletters, campañas de email y plantillas transaccionales.
