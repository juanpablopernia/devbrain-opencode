# Flujo de trabajo

## Proyecto nuevo

1. Inicializar proyecto:

```bash
/path/to/devbrain-opencode-v2/bin/init-project.sh .
```

2. Abrir OpenCode:

```bash
opencode
```

3. Levantar requisitos:

```text
/intake
```

4. Planificar arquitectura:

```text
/plan
```

5. Implementar:

```text
/implement crear estructura base
```

6. Revisar:

```text
/review
```

7. Documentar:

```text
/document
```

## Regla de oro

No implementar proyectos nuevos sin ficha técnica.
