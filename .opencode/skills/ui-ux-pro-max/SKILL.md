# UI/UX Pro Max

Skill para elevar calidad visual y de experiencia en interfaces web o móviles.

## Cuándo usarla

- Diseño de pantallas, landing pages o dashboards.
- Refactor visual de componentes.
- Decisiones de color, tipografía, espaciado y jerarquía.
- Revisión de UX, accesibilidad y consistencia.

## Prioridades

1. Accesibilidad.
2. Claridad de interacción.
3. Performance percibida.
4. Dirección visual coherente.
5. Responsive real.

## Reglas

- Cada pantalla debe tener una jerarquía visual clara.
- Usa una dirección estética intencional; evita UI genérica.
- Mantén consistencia entre navegación, spacing, radios, sombras y tipografía.
- Las acciones primarias deben ser obvias.
- Los formularios deben mostrar labels, errores inline y feedback de carga.
- Considera siempre móvil, tablet y escritorio.

## Checklist rápido

- ¿El contraste y foco son correctos?
- ¿La pantalla comunica claramente la acción principal?
- ¿Los estados loading/error/empty/success están diseñados?
- ¿La densidad visual coincide con el tipo de producto?
- ¿La interfaz parece pensada o solo ensamblada?

## Complementos recomendados

- Usa `frontend` para implementación.
- Usa `modern-web-guidance` para patrones web modernos.
- Usa `web-animation-design` si la experiencia depende de motion.
